I'm a Systems Development student from Brazil focused on **backend development, databases, and system reliability**.

I enjoy working close to the core of applications — designing **data models**, writing **efficient SQL**, and building services that move data safely between systems.

# Featured Project

## Sangue Solidário

A platform designed to **connect blood donors with hospitals and blood centers across Brazil**.

Blood donation is often urgent and difficult to coordinate.  
**Sangue Solidário** aims to simplify this process by providing a platform that connects donors directly with healthcare institutions.

Key concepts explored in the project:

- Donor registration and matching system
- Blood type availability tracking
- Hospital and blood center integration
- Scalable backend architecture
- Reliable relational database design
